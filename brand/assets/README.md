# Qooee Brand Assets

Use semantic filenames in product and marketing work.

## Wordmarks

- `logo/qooee-wordmark-primary.svg`
- `logo/qooee-wordmark-inverse.svg`
- `logo/qooee-wordmark-primary.png`
- `logo/qooee-wordmark-inverse.png`

Use the primary wordmark on Isabelline, white, and light warm neutrals. Use the inverse wordmark on navy or another dark surface.

The SVG wordmarks currently contain live text. Treat them as supplied source assets until outlined production exports are created.

## Fonts

- `fonts/CreteRound-Regular.ttf`
- `fonts/OFL.txt`

Crete Round is used only to support the supplied wordmark source files. It should not become the general Qooee product or marketing typeface.

Official source: [Google Fonts: Crete Round](https://fonts.google.com/specimen/Crete+Round).

To install the font locally on macOS:

```sh
cp brand/assets/fonts/CreteRound-Regular.ttf ~/Library/Fonts/
fc-cache -f ~/Library/Fonts
```

## Icon Marks

- `icon/qooee-icon-primary.svg`
- `icon/qooee-icon-inverse.svg`
- `icon/qooee-app-icon-primary.svg`
- `icon/qooee-app-icon-dark.svg`

Use the transparent icon marks for compact UI, favicons, loading states, and avatar-like contexts where the brand is already known. Use the app icon SVGs as direction assets for platform app icon production.

## Do Not

- Recolour the Honey Yellow letters.
- Substitute pure black for Rich Black FOGRA 29.
- Substitute pure white for Isabelline unless a technical constraint requires it.
- Add gradients, shadows, rotation, or distortion.
- Place the full wordmark below 120 px wide on screen.
