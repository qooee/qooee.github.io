# Qooee Palettes

These `.clr` files are the supplied Apple colour palettes and remain the source of truth for brand colour values.

## Files

- `qooee-primary.clr`: Honey Yellow, Rich Black FOGRA 29, Isabelline, Ruby Red.
- `qooee-secondary.clr`: Steel Blue, Silver Pink, Xanadu.
- `qooee-tertiary.clr`: Honey Yellow tints, Ruby Red tints, Steel Blue tints, Platinum, Light Gray.

The Markdown design system and token files repeat these values for implementation convenience. If a value ever differs, treat these source palettes as canonical and correct the derived files.
